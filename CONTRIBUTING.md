# Contributing to claude-code-power8

Thank you for your interest in contributing to this project!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/claude-code-power8.git`
3. Create a new branch: `git checkout -b your-feature-name`

## Making Changes

1. Make your changes in your feature branch
2. Test your changes thoroughly
3. Commit your changes: `git commit -m "Description of your changes"`
4. Push to your fork: `git push origin your-feature-name`

## Submitting a Pull Request

1. Create a pull request from your fork to the main repository
2. Fill in the PR template with all relevant details
3. Wait for review and address any feedback

## Code Standards

- Follow existing code style and conventions
- Write clear, descriptive commit messages
- Include comments for complex logic

## Questions?

If you have questions, feel free to open an issue for discussion.

## Recognition

Contributors will be recognized in the project README.
